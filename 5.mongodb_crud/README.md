# Week 16 — MongoDB CRUD Exploration

Personal exploration project for practicing MongoDB operations using employee data. Minimal structure, no K8s.

---

## What's here

- `app/connection.py` — MongoDB connection setup
- `app/dal.py` — Data access layer: CRUD operations
- `app/main.py` — FastAPI entry point
- `app/routs.py` — FastAPI route definitions
- `app/json_to_mongo.py` — Script to load JSON data into MongoDB
- `app/local_load.py` — Local data loading helper
- `employee_data_advanced.json` — Sample dataset (~88KB, employee records)
- `requirements.txt` — Python dependencies

**Stack:** FastAPI · MongoDB · PyMongo
